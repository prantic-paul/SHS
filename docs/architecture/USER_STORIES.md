### **User Story : **
User Story 1:
As a Doctor 
I want to register and login into the app
So that I can give treatement to the patient

Acceptance Criteria 1:
Given I am on Registration Form, when I enter valid details , then my doctor account is created


User Story 2:
As a Doctor 
I want to write blog
So that i can create awareness among the people

Acceptance Criteria 2:
Given I am in a blog form , when I wrote the blog and click on upload , then the blog have been visible for the general people


User Story 3:
As Doctor 
I want to see previous report of patient
So that I can give a better treatement

Acceptance Criteria 3:
Given A button show previous record of the specific patient , When I clicked on it with valid user id, then it shows me previous record of the patient 


User Story 4:
As a Doctor 
I want to create and edit a digital prescribetion for the patient
So that I can continue my treatement for the patient

Acceptance Criteria 4:
Giveen A button new prescribetion for the specific patient , when i click on it it shows me a form and  write a prescribetion ,then the prescribetion is visible for the patient 


User Story 5:
As a general user
I want use a chatbot
So that I can get the appropiate suggetion for doctors


Acceptance Criteria 5:
Given a text box for chat , when i type something and click suggest button , then it shows me some doctors profile


User Story 6:
As a general user
I want to appoint a doctor
So that I can get treatement from the doctor

Acceptance Criteria 6:
Given a button at the doctor profile , when i click there it ask me for time and date , then it has been  booked appointment for me
